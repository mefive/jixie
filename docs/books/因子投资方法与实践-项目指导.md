# 《因子投资：方法与实践》对 jixie 的业务指导

> 来源:石川、刘洋溢、连祥斌,电子工业出版社,2020。原书见同目录 EPUB。
> 本文是面向 jixie 的提炼,不是全书摘要,也不替代 `ROADMAP.md` 和 `docs/design/` 中已经拍板的设计。

## 结论

这本书对 jixie **有直接且长期的业务指导作用**,主要覆盖因子研究和投资组合实践,而非软件架构。它支持项目当前的核心定位:不做“因子动物园”的搬运工,而是把公开因子放到 A 股、真实数据口径和真实交易约束下重新验证。

jixie 已经吸收了书中相当一部分关键思想:PIT 财务数据、后复权研究价格、截面排序和 Rank IC、市值/行业中性化、费后分析、因子相关性、不可变研究台账、样本外保留期规划、多重检验提示规划、A 股成交限制和滑点冲击。后续重点不是另起炉灶,而是补齐研究纪律和统计验证。

## 长期业务原则

### 1. 先有假设,再做检验

- 每个候选因子必须说明经济或行为逻辑、预期方向、信息为何没有立即进入价格,不能只因为样本内 IC 或收益好看就收录。
- 低 p 值、高 IC、高 Sharpe 都只是证据的一部分,不能代替因果解释、可交易性和样本外表现。
- 无效结果同样有价值。证伪的因子要留台账,避免换参数或换名字后重复挖掘。

这与项目“验证者 / 实现者,不是发现者”以及预置因子三道门一致。

### 2. 因子验证要形成证据链

最低限度的验证顺序:

1. 确认数据、股票池、时间语义和预期方向;
2. 看 Rank IC、IC 稳定性和十分位分层是否大体单调;
3. 同时看等权和市值加权,识别小盘暴露;
4. 做市值 / 行业中性化,确认不是已知风格换皮;
5. 看衰减、换手、费后收益和容量;
6. 在未参与开发的保留段只验证一次;
7. 需要更强结论时,再用 Fama-MacBeth、时序 alpha 和稳健标准误等回归方法验证增量解释力。

单一漂亮指标不能替代整条证据链。多空组合在 A 股主要是研究诊断工具,不是默认可执行策略;最终仍要进入多头策略回测验证。

### 3. 时间语义比公式更重要

- 财务数据必须按公告日 PIT 门控,只能使用当时已经公开的最新版本。报告期不等于可用日期。
- 量价研究使用后复权序列保持收益连续;真实成交、信号展示和订单数量使用不复权真实价格。
- 不同字段不能统一“前填”:财务存量可 as-of,资金流等日流量缺失即缺失,停牌日如何处理取决于指标含义。
- 动量的基准日价格可以按明确规则对齐;波动率、beta 等不能把停牌日机械填成零收益,否则会系统性低估风险。
- 窗口指标应声明最少有效样本数。书中的经验基线是窗口的三分之二,jixie 可按指标覆盖测试后调整,但不能默默用极少样本计算。

### 4. 股票池必须是当时可投资的股票池

- 不能用今天的成分或存续股票回看历史,必须保留退市股并按历史状态选股,避免幸存者偏差。
- 上市不足一年、ST / 风险警示、待退市、净资产为负、长期停牌和极低流动性应成为显式、可说明的股票池规则。
- 研究股票池与实际策略股票池可以不同,但报告必须展示过滤规则和样本数量,不能把过滤藏在实现里。
- 金融股是否剔除不是全局真理。对普适量价因子可保留;涉及会计口径的基本面因子应做行业适用性判断或行业中性化。

### 5. 异常值处理必须作用在正确对象上

- 因子暴露和未来收益都可能有异常值,两者应分别定义处理规则。
- Rank IC 对极值相对稳健,但中性化 OLS、z-score 和参数化收益模型会被极端因子值显著影响。
- 默认 1% / 99% 缩尾可作为基线,但不能无脑用于所有因子。异常换手等变量可能需要 MAD 或业务边界;报告应记录实际方法。
- 不把真实的涨跌停或上市首日波动简单当脏数据。研究清洗、可成交性和引擎成交规则是三件不同的事。

### 6. 纸面收益必须经过可交易性检验

- 信息衰减决定调仓频率;调仓频率又决定换手、成本和容量,三者必须一起看。
- 成本至少包括佣金、卖出印花税、价差 / 滑点和随订单额占成交额上升的冲击成本。
- 低流动性溢价尤其容易“看得见、拿不到”;不能只用固定费率掩盖容量问题。
- 因子研究报告的费后多空用于横向比较生存力;策略引擎负责 T+1、停牌、涨跌停、整手、真实资金规模和成交失败等最终验证。

### 7. 样本外和多重检验是产品能力,不是文案提醒

- 因子 / 参数每试一次都应进入台账,包括失败和被否决的结果。
- 默认留出最近 18 个月作为 holdout;看过并据此调整后,它就不再是样本外。
- 台账至少展示累计检验次数和假阳性提示。候选规模扩大后,优先采用 FDR 类方法,而不是只盯传统 `t > 2`。
- 公开发表后的因子溢价通常衰减,因此文献结论只能作为先验,不能直接当成本地预期收益。

### 8. 简单组合是复杂模型的强制基线

- 多因子首先使用截面 rank / z-score 标准化后等权合成;按主题分层合成优于把大量相似变量一次性相加。
- IC 加权、风险平价、均值-方差优化和机器学习都必须与等权基线比较,并用样本外增益证明复杂度值得。
- 因子择时关系不稳定、容易数据窥探,考虑成本后常难战胜长期等权。jixie 不应把动态因子择时作为近期主线。
- 预期收益和协方差估计误差会让优化权重剧烈变化。没有明确投资组合需求前,不提前建设 Barra 或通用优化器。

### 9. 因子研究不替代基本面判断

因子能把可重复的公司特征机械化,但会丢失商业模式、治理、竞争格局和事件上下文。jixie 的定位应是提供可复现的量化证据和中性工具,不把因子分数包装成对公司“真实价值”的完整判断。

## 对路线图的指导优先级

### P0:先补研究纪律

1. 优先完成 ROADMAP 3.6a:在已实现的报告台账上补研究卡、一次性 holdout 和多重检验提示。
2. 把股票池过滤、因子值异常处理、窗口最少有效样本写成报告可见的分析参数 / 元数据。
3. 数据质量审计覆盖退市股、历史 ST 状态、财务 PIT 和缺失窗口,避免研究层靠静默过滤兜底。

### P1:补强统计验证

在因子候选增多后,增加 Newey-West 稳健 t 值、Fama-MacBeth 截面回归和对已有因子模型的 alpha 检验。它们用于回答“控制常见特征后是否仍有增量”,不取代 Rank IC 和分层图。

### P2:多因子先简单后复杂

实现顺序保持:低相关因子筛选 → rank 标准化等权 → IC 加权对照 → 多头策略回测。只有等权组合样本外有效后,才评估风险模型、约束优化或 ML。

### 明确不因本书而提前做

- 不批量导入数百个因子;
- 不把多空研究组合当作 A 股可直接执行策略;
- 不照搬书中零成本实证口径;
- 不把所有历史日收益硬压到 ±10%:创业板 / 科创板、北交所、ST、上市初期等限制不同,应以历史板块规则和真实 `StkLimit` 数据为准;
- 不因书中介绍了 Barra、GMM、组合优化就提前建设重型数学基础设施;
- 不做因子择时优先于静态分散组合。
